About This Project

Project Overview

This application allows users to detect plant leaf diseases using a trained deep learning model. It provides instant remedies by fetching data from a Firebase Realtime Database, helping farmers, gardeners, and researchers mitigate crop loss efficiently.

Technologies Used

  • Python, FastAPI (Backend)
  • Firebase Realtime Database (Cloud Storage)
  • FastAI with ResNet-34 (Deep Learning)
  • FPDF (PDF Generation)
  • HTML/CSS/JavaScript (Frontend)

Supported Crops

  • Tomato
  • Potato
  • Apple
  • Grapes
  • (More can be added in the future)

How It Works

  1. User uploads an image of a plant leaf.
  2. The image is processed and sent to the FastAPI backend.
  3. The backend uses a pre-trained model to classify the disease.
  4. Remedies are fetched from Firebase and displayed to the user.
  5. Users can download a PDF report of the analysis.

Motivation

This project aims to assist farmers and agricultural specialists by providing a reliable and accessible platform for identifying plant diseases and suggesting appropriate remedies. With food security being a major concern globally, early detection and prevention of plant diseases can save crops and improve yield quality.

The system combines the power of deep learning (ResNet-34) with cloud technologies like Firebase and modern asynchronous web frameworks to deliver fast and accurate results. It is designed with usability and accessibility in mind, allowing users to interact with a clean web interface from desktop or mobile.

Our Mission

Our mission is to bridge the gap between machine learning research and real-world agricultural needs. By leveraging AI and cloud computing, we aim to empower communities with tools that are simple, free, and effective in combating plant health issues.
